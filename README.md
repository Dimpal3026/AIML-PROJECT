# AIML-PROJECT
This project, titled Student Performance Prediction & Smart Study Recommendation System, focuses on applying Machine Learning techniques to analyze and predict student academic performance. The system uses important factors such as study hours, attendance, sleep patterns, and previous scores to generate accurate predictions of a student’s final performance.

The project aims to address the common problem of students being unaware of their academic progress until it is too late. By providing early predictions, the system helps students identify their strengths and weaknesses in advance. In addition to prediction, the project also includes a smart recommendation system that offers personalized study suggestions based on the predicted performance level.

The model is developed using Python and basic Machine Learning algorithms like Linear Regression, along with libraries such as Pandas and Scikit-learn for data processing and model training. A rule-based logic system is also integrated to generate meaningful recommendations that guide students toward better study habits.

Overall, this project demonstrates how Artificial Intelligence and Machine Learning can be effectively used in the education sector to improve student performance, support decision-making, and promote smarter learning strategies.
